#### Router BGP Device Configuration

```eos
!
router bgp 65103
   router-id 10.0.0.245
   maximum-paths 16 ecmp 16
   no bgp default ipv4-unicast
   neighbor AUTOVPN_EDGE peer group
   neighbor AUTOVPN_EDGE remote-as 65000
   neighbor AUTOVPN_EDGE bfd
   neighbor AUTOVPN_EDGE password 7 <removed>
   neighbor AUTOVPN_EDGE send-community
   neighbor AUTOVPN_EDGE maximum-routes 12000
   neighbor 10.250.252.1 peer group AUTOVPN_EDGE
   neighbor 10.250.252.1 description S-2A
   neighbor 10.250.253.1 peer group AUTOVPN_EDGE
   neighbor 10.250.253.1 description S-2B
   redistribute connected
   !
   address-family ipv4
      neighbor AUTOVPN_EDGE activate
```
